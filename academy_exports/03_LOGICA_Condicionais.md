## Aula 1: O bloco If e Else

### 📖 Explicação Teórica
O `if` é o coração da lógica. Ele permite que o código execute apenas se uma condição for verdadeira.

```javascript
if (temperatura > 30) {
  console.log("Está calor!");
} else {
  console.log("Está fresco.");
}
```

### 💻 Exemplos de Código
```javascript
let saldo = 50;
let compra = 100;

if (saldo >= compra) {
  console.log("Venda aprovada!");
} else {
  console.log("Saldo insuficiente.");
}
```
