## Aula 1: Operadores Aritméticos

### 📖 Explicação Teórica
A programação é feita de cálculos. Os operadores básicos são:
- `+` (Soma)
- `-` (Subtração)
- `*` (Multiplicação)
- `/` (Divisão)
- `%` (Resto da divisão - Modulo)
- `**` (Potência)

```javascript
let total = 10 + 5 * 2; // Resultado: 20 (segue a ordem matemática)
```

### 💻 Exemplos de Código
```javascript
let preco = 100;
let desconto = 15;
let final = preco - desconto;
console.log("Preço com desconto: " + final);
```

## Aula 2: Comparações e Booleanos

### 📖 Explicação Teórica
Para tomar decisões, precisamos comparar valores:
- `==` (Igual a)
- `!=` (Diferente de)
- `>` (Maior que)
- `<` (Menor que)
- `>=` (Maior ou igual)

### 💻 Exemplos de Código
```javascript
let idade = 18;
let podeEntrar = idade >= 18;
console.log("Pode entrar? " + podeEntrar);
```
