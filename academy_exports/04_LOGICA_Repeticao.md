## Aula 1: While e For

### 📖 Explicação Teórica
Loops permitem executar o mesmo bloco de código várias vezes.
- **While:** Repete enquanto a condição for verdadeira.
- **For:** Repete uma quantidade específica de vezes.

```javascript
for (let i = 1; i <= 5; i++) {
  console.log("Passo " + i);
}
```
