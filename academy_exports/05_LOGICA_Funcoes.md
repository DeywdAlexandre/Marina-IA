## Aula 1: Declarando Funções

### 📖 Explicação Teórica
Uma função é um "verbo". Ela executa uma ação quando chamada.

```javascript
function saudar(nome) {
  return "Olá, " + nome + "!";
}
console.log(saudar("Marina"));
```
